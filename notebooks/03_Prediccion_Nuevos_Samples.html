# %% Crear secuencias
N_STEPS = 7  # Debe coincidir con el entrenamiento

def create_sequences(X_data, time_steps=7):
    """Crea secuencias temporales para el modelo GRU"""
    Xs = []
    for i in range(len(X_data) - time_steps):
        Xs.append(X_data[i : (i + time_steps)])
    return np.array(Xs)

X_nuevos_seq = create_sequences(X_nuevos_scaled, N_STEPS)

print(f" Secuencias creadas: {X_nuevos_seq.shape}")

8. Conclusiones

El modelo de Red Neuronal Recurrente, optimizado mediante Grid Search, demostr un rendimiento excepcional al capturar la tendencia general de los datos.

  • Alto Rendimiento: Alcanzamos un coeficiente de determinacin ($R^2$) de 0.9653 en el conjunto de prueba, lo que confirma que el modelo explica exitosamente el $\mathbf{96.5\%}$ de la variabilidad del stock real. El error promedio (MAE) de $\mathbf{58.8}$ unidades es bajo en el contexto de los valores de stock mximos.
  • reas de Enfoque: El anlisis detallado de los residuales y los errores por ejemplo (con errores porcentuales altos, de hasta $\mathbf{58\%}$ en valores de stock bajos) revel una inconsistencia y sesgo sistemtico en el rango de baja magnitud. El modelo tiende a sobreestimar o subestimar estos valores de forma errtica.

Planes Futuros: Para alcanzar una precisin perfecta en todo el rango de datos, el trabajo futuro se enfocar en:

  1. Robustez: Implementar una transformacin logartmica en los datos de entrada para estabilizar la varianza y dar mayor peso a los valores pequeos durante el entrenamiento.
  2. Alineacin de Mtrica: Explorar el uso de una funcin de prdida que penalice los errores en funcin de su porcentaje (MAPE) para mitigar el sesgo detectado.
